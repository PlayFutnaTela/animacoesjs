# Steering: Anime.js Animation Expert

## Visão Geral

Esta steering file orienta o uso do Anime.js para criação de animações web de alta qualidade.

## Quando Usar

Esta steering deve ser ativada quando:
- O usuário pedir ajuda com animações Anime.js
- O usuário quiser criar animações para seu projeto
- O usuário precisar de recomendações de modelos de animação
- O usuário quiser otimizar performance de animações

## Instruções

1. **Identifique o objetivo da animação**
2. **Recomende o tipo de animação apropriado**
3. **Sugira os melhores easings para o cenário**
4. **Oriente sobre performance e boas práticas**
5. **Forneça exemplos de código completos**
